Il est la base constitutive de toutes les pages web qui existent. Il est souvent utilisé avec du CSS (Cascading Style Sheets). Le cour sur le css est disponible dans les Courses, nous n'allons pas en parler ici. le HTML est un langage très simple qui permet à chacun, moyennant quelques petites heures d'apprentissage, de diffuser des documents au monde entier. On a en fait pas encore trouvé plus simple, et les langages ressemblant à HTML se sont multipliés comme la peste, mais pour le bonheur de nombreuses personnes. HISTORIQUE

Documentation

L'hypertexte Markup Language ou généralement abrégée «HTML» est un langage de balisages permettant la création d'un site. Dans les années 1989-1990, un informaticien Tim Berners-Lee invente le Worl Wide Web. Le HTML est une des trois inventions issus du WWW avec le Hypertext Transfer Protocol ( HTTP ) et les adresses web. La première version du HTML comprend le titre du document, la structuration du texte en titres, sous-titres, listes ou texte brut, les hyperliens mais aussi un principe de référencement. Tim Berners Lee base le html sur le SGML (Standard Generalized Markup Language) qui est un langage généralisé des balises standardisées afin d'attirer les entreprise sur le html. À la fin des années 1993, le HTML qui est alors a sa version 1 voit un effet de normalisation. On l'appelle dorénavant HTML +. Malheureusement, jusqu'à la fin des années 1990 le html est défini par les implémentations des navigateurs. L'arrivé du navigateur NCSA Mosaic amène deux éléments majeurs : l'insertion des images et l'introduction des formulaires. Le 13 novembre 1993, l'apparition du nouveau navigateur Netscape Navigator 0.9 permet l'ajout de nombreux éléments : attributs de texte, centrage etc. Maintenant le html et les navigateurs prennent en compte le design et même élargissent ses capacités notamment avec le CSS. Le protocole décrivant HTML 2.0 est finalisé à la fin de l'année 1995. Il décrit le HTML d'avant juin 1994 c'est à dire sans les nombreuses mise à jour dû au navigateur Netscape Navigator 0.9. Dan Conelly est le principal rédacteur de ce document. En décembre 1997 le W3C ( World Wide Web Consortium ) fondé par Tim Berners-Lee en 1994 publie le HTML4.0. Les principales nouveautés sont la séparation remarquable entre la structure et la présentation du document mais aussi la standardisation des éléments de style et des scripts. D' autres éléments sont ajoutés notamment la balise framset, transitional ou encore strict. Dans les années 2000, les informaticiens préfèrent s'atteler au XHTML et oublient le HTML. Ce n'est qu'en mars 2007 que la W3C décide de relancer le développement du html, avec à la tête des recherches Chris Wilson (Microsoft).

Les balises et les attributs

Documentation

LES BALISES
Le html est un language de balisage car utilisant des balises. les balises dans le fichier html sont de petit mots qui sont écrit entre des chevrons < >. Ils ne sont pas visible sur la page web. À quoi elles servent ces balises?
Elles indiquent la nature de leurs contenu, c'est à dire qu'elles siblent des éléments précis dans la pages web.
Ex: Des Paragraphes, des images, des titres de pages etc...
Il existe deux types de balises,
Les balises en paires
Ces balises peuvent contenir, du texte, des images ou encore d'autres balises
HTML <p>Je suis un paragraphe</p> Les balises orphelines
Ces balises sont généralements utilisés pour insérer des éléments dans la page web
HTML <image/> Ceci veut dire à l'ordinateur d'insérer dans la page web une images
LES ATTRIBUTS
Les attributs sont des parametres ajoutés aux balises pour les identifier, pour leurs ajouter des fonctionnalités ou pour les modifier. Les attributs se placent après le nom de la balise ouvrante et a le plus souvent une valeur, comme ceci :
HTML <balise attribut="valeur">

HTML <!DOCTYPE html>

Role de chaque balise de bases

Documentation

Le <!DOCTYPE html> au début de notre code indique qu'il s'agit d'un document HTML
Les deux balises ouvrante et fermante <html> </html> englobent tout le contenu notre code html
Dans l'entete <head> nous avons des informations(l'encodage,documents liés,...) qui ne sont pas directement visibles dans la page web.
Le <meta charset="UTF-8"> définit l'encodage du document html. l'encodage est comme vous pouvez le voir en UTF-8, il est plus conseillé. Comme vous auriez déja deviner, c'est la balise <title></title> qui définit le titre de document html. Le title du document apparait en haut dans le navigateur.
Enfin juste en bas de la balise fermante de head, nous avons la balise <body></body>. La partie visible de notre page web est contenu dans cette balise.

LES LISTES EN HTML
Il existe deux types de listes en html, Les listes ordonnées et Les listes non ordonnées
Pour faire une liste ordonnées, on utilise la balise <ol></ol> et ensuite pour définir les termes de la liste on utilise la balise <li></li>
Pour faire une liste non ordonnées, on utilise la balise <ul></ul> et ensuite pour définir les termes de la liste on utilise la balise <li></li>
Nous allons définir une liste d'aliment. HTML <p>Listes ordonnées</p>
<ol>
<li>Orange</li>
<li>Banane</li>
<li>Ananas</li>
</ol>
<p>Listes non ordonnées</p>
<ul>
<li>Orange</li>
<li>Banane</li>
<li>Ananas</li>
</ul>


Ici la liste ordonnées est définit avec des nombres 1,2,3...
Mais sachez que vous pouvez changez cet affichage. Avec le CSS que nous veront plus tard, vous serez en mesure de le définit en, par exemple A,B,C...

Outre toutes ces balises dont nous venons de parler, il y a deux balise appelées des balises universel qu'on utilise souvent, il s'agit des balise <div></div> et <span></span>
div est une balise Block et span est une balise inline

LES LIENS ET LES IMAGES

Documentation

LES LIENS
Vous avez l'habitude, une fois sur les sites web de clicker sur de petits mots, phrase, images en miniature... pour accéder à d'autre page de ce meme site web ou de naviguer entre plusieurs. Ce sont les liens qui jouent ce role et qui vous permettent de naviguer entre les pages web. Pour faire un lien, la balise que nous allons utiliser est très simple à retenir; c'est la balise<a></a>, Il faut cependant lui ajouter un attribut, href, pour indiquer vers quelle page le lien doit conduire.
HTML <a href="http://www.sololearn.com"> je suis un lien </a>

La balise <a href=""> peut tout contenir, que ca soit du texte, des images, des box...
LES IMAGES
Pour insérer des images dans votre page web, il vous suffit utiliser la balise <img src=""> et de lui ajouter un url comme ceci:
HTML <img src="superMario.png">

Attention, ici l'image en question était dans le meme dossier que mon fichier html si ce n'était pas le cas, il va falloir changé le code html.
